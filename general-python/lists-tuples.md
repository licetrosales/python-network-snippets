## Python Lists vs Tuples

- **List**: Mutable, ordered collection. Can hold different data types, including nested lists. Defined using square brackets `[]`.

- **Tuple**: Immutable, ordered collection. Also supports different data types. Defined using commas, often enclosed in parentheses `()`.

**Python built-in functions** and **list methods** for handling sequences like lists and tuples.

### üîß Built-in Functions
- `len()`, `max()`, `min()`: Analyze sequence size and values.
- `list(seq)`, `tuple(seq)`: Convert between list and tuple types.
- `cmp()`: Compare two sequences (Python 2 only, not in Python 3).

### üß∞ Common List Methods
- Add/remove: `append()`, `remove()`, `pop()`, `insert()`, `extend()`
- Search/count: `index()`, `count()`
- Modify: `sort()`, `reverse()`, `clear()`, `copy()`
